## Image Guidelines

When adding images to this directory:

- Use PNG format for screenshots and logo
- Compress images to keep file sizes reasonable
- Use descriptive filenames
- Recommended dimensions:
  - Logo: 400x400px (transparent background)
  - Screenshots: 1170x2532px (iPhone 14 Pro Max) or similar
